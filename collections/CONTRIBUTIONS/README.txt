# READ THIS TO CONTRIBUTE!

This is the central collection general comments and accepting new contributions for this workspace. Providing a single doorway for public users of this workspace to contribute to the available APIs within this workspace. 

To leave a general comment or ask a question, you can click on the comment link for this pane. If you have feedback on an existing API make sure you leave a comment on that individual API collection or OpenAPI.

To add a new collection, you can fork this collection, add a folder for your collection and add your collection under there, then submit a pull request. If the merge is accepted, we'll merge, then drag the folder into it's own collection.
