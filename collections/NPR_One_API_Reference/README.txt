NPR One is a smart application that brings the best of NPR and Member Station programming, newscasts, podcasts, and stories together to create a new experience for listening. It provides an editor-curated and localized mobile listening experience based on the content the listener chooses, likes, shares, and enjoys. The API provides all of the content and customization in a simple, structured way that is easy for applicationndevelopers to implement.
