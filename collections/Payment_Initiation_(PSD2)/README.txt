The Payment Initiation API is used to send a payment orders and check their processing status. The API can be used by licensed third party service providers to access ABN AMRO accounts. An account can only be accessed when ABN AMRO's account holder authorizes this through the consent application.
**New in version 1.1.1:**
- Added Content-Length header for PUT method
- Textual improvements
- Added delete method for standing payment orders
- Added X-Request-ID Header Support
- Improved error messages
